# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-25

### Added

- 准备发布首个稳定版本。

### Changed

- 使用 Vite 构建 MCP Server，并发布生成后的服务入口。
- MCP 冒烟测试改为检查构建后的产物。

## [0.1.0] - 2026-06-25

### Added

- 初始发布高德天气 MCP Server。
- 支持实况天气和天气预报查询工具。
- 支持通过 stdio 与 MCP 客户端通信。
